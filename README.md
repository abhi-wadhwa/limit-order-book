# limit-order-book

Limit order book simulator with price-time priority matching, Avellaneda-Stoikov market making, execution algorithms, and microstructure analytics.

## Quickstart

```bash
git clone https://github.com/abhi-wadhwa/limit-order-book.git
cd limit-order-book
make install
make run
make test
```

See `docs/` for the math.
